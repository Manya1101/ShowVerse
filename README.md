# 🎬 ShowVerse

ShowVerse is a **full-stack MERN application** that allows users to discover, explore, and manage movies & TV shows with a clean UI, secure authentication, and scalable backend architecture.  
It integrates **Clerk authentication**, **Inngest background jobs**, and an external movie API to deliver a production-ready experience.

---

## 🚀 Features

- RESTful APIs for movies & TV shows
- Secure authentication using **Clerk**
- Background jobs & workflows using **Inngest**
- MongoDB-based data persistence
- Clean and scalable Express architecture

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB & Mongoose
- React(Vite)
- Clerk Authentication (`@clerk/express`)
- Inngest (`inngest/express`)
- dotenv & CORS

---

## 🔐 Authentication

- User authentication handled via **Clerk**
- Clerk middleware protects private API routes
- No passwords are stored in the database
